# TeamViewer
TeamViewer is a helpful tool used in IT to remoto into and manage user computer it's used for:
- Troubleshooting issues
- Installing Software
- Configuring settings
- Providing technical assistance without being physicall present
## Setting up TeamViewer 
1. Install TeamViewer (desktop client)
<img src="https://github.com/MahTimbs/IT-Documentation/blob/main/Remote%20Connection/Images/T1.png?raw=true" height="800" width="600" />
 
## Creating a session
1. You can share a link or a code
2. send an invite
- The user needs to type in the session code to connect remotely or share their ID and password from the TeamViewer to the support technician
<img src="https://github.com/MahTimbs/IT-Documentation/blob/main/Remote%20Connection/Images/T2.png?raw=true" height="800" width="600" />

## File Transfer and Remote Control
- From the top menu > **File & Extras**, you can move files from your PC to the remote computer
- Make sure that you remember you need to choose a place on their PC as well for this to work
<img src="https://github.com/MahTimbs/IT-Documentation/blob/main/Remote%20Connection/Images/T3.png?raw=true" height="800" width="600" />

## Joining a session
1. The user adds the given session code
